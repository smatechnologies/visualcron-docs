# Start Guide Wizard

The Start Guide is a first-run wizard that appears after installation. It walks you through the key initial setup steps for VisualCron.

You can relaunch it at any time from **File > Settings > Start Guide**.

## What the wizard covers

The Start Guide presents a series of screens that introduce:

1. **Server connection** — connecting the Client to the local or remote Server
2. **License activation** — entering or requesting a license key
3. **Initial settings** — basic server configuration options

## After the wizard

Once you have completed the Start Guide:

- Connect to your server and change the default admin password (**Server > Main Settings > User Permissions**)
- Create your [first job](your-first-job)
- Explore [Core Concepts](../core-concepts/jobs-and-tasks) to understand triggers, conditions, and variables
