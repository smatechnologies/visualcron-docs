## Overview

The **Tools** tab is a part of the main menu. It groups a number of windows that help you explore, analyze and maintain your VisualCron setup:

* **Explore** - [File Explorer](explore-file-explorer), [SQL Explorer](explore-sql-explorer), [Connection Explorer](explore-connection-explorer), [Server monitor](server-monitor) and [Task Manager](task-manager).
* **Objects** - [Object relations](object-relations), [Object search](object-search), [Object replace/merge](object-replace-merge) and [Audit log](objects-audit-log).
* **Calendar** - [Calendar](schedule-calendar).
* **Flow chart** - [Flow chart](flow-chart-flow-chart).
* **Gantt chart** - [Gantt chart](gantt-chart).
