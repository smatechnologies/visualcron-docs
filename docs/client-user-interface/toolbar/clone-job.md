## Clone Job

A marked Job in the Server/Job/Task grid can be cloned by clicking toolbar the Clone Job option. See [clone Job](../server-group-job-task-area/job-list/clone-job) and the [Main - Jobs](../server/jobs-jobs) section for more information.
