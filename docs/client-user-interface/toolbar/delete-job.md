## Delete Job

A marked Job in the Server/Job/Task grid can be deleted by clicking toolbar the *Delete Job* option.

A confirmation prompt is shown before the Job is removed. If the Job is used by other objects, the prompt also reports how many objects reference it and suggests opening the Object relations window for more details.
