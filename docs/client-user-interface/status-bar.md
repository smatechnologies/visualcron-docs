## Overview

The status bar is located in the bottom of the client window. From left to right it shows:

* [Client events](client-events) - notifications about Jobs that start, complete or fail.
* [Server name](server-name) - the server the client is currently connected to.
* [Server time](server-time) - the current time on the connected server.
* **Jobs** - the number of Jobs on the connected server.

A progress bar is also shown to the right while connecting to a server.
