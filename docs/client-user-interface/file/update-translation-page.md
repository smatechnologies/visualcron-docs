## Translation - Update Translation Files

The main menu **File > Translation > Update translation files** option rebuilds the local language files for the currently selected client language. It scans the Client for captions, saves the language files and reloads the current language, so that new or updated translations take effect.

If your language is missing in VisualCron or if a specific VisualCron caption is missing in your language, and you want to contribute with translation work, please contact [VisualCron support](https://www.visualcron.com/contact.aspx).
