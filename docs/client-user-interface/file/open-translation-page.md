## Translation - Open Translation Page

The main menu **File > Translation > Open translation page** option opens the VisualCron translation page (https://www.visualcron.com/translation/translationstart.aspx) in your web browser. This is where you can contribute translation work, for example if your language is missing in VisualCron or if a specific VisualCron caption is missing in your language.

If you need help getting started, please contact [VisualCron support](https://www.visualcron.com/contact.aspx).
