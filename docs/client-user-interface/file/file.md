## File

The **File** tab is a part of the main menu. It groups the following options:

* **Import/Export** - [Import settings](import-settings), [Export settings](export-settings) and [Task scheduler](task-scheduler).
* **Updates** - [Check for updates](check-for-updates).
* **Settings** - [Client settings](client-settings) and [Start guide](start-guide).
* **Servers** - [Manage servers](manage-servers).
* **Translation** - [Open translation page](open-translation-page) and [Update translation files](update-translation-page).
