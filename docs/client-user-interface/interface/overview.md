## Overview

The Interface tab is a part of the main menu.

![](../../../static/img/Client%20User%20Interface/Main%20Menu/Interface/Interface.png)

The current servers, Jobs and Tasks are listed in the Server/Job/Task grid.
 
The format of the Job/Task listing is controlled from the main menu **Interface** tab, which is divided into three groups:

* **Columns** - [Group jobs](columns-group-jobs), [Set columns](columns-set-columns) and [Expand all](columns-expand-all).
* **Filter** - [Grid filter](filter-grid-filter) and [Clear filter](filter-clear-filter).
* **Theme** - a drop down list for switching the client theme between *Dark Blue*, *Dark Grey* and *Light Grey*.

The Job/Task list can be customized by filtering a specific column and setting the columns to be visible/invisible.
 
A view filter can be defined for one column by a match criteria.
 
The number and the order of columns to view in the Job list can be modified.
