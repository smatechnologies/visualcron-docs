## Columns - Group Jobs

With the main menu **Interface > Columns > Group jobs** option, it is possible to group the Server/Group/Job/Task grid by the Job group. When the option is selected, the Jobs are grouped together by their group; when it is not selected, the grid is shown without grouping.
