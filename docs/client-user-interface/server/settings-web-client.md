## Settings - Web Client

In the main menu **Server > Main settings > Settings** dialog, there are a set of important setting groups/tabs. In this tab, the web client settings are managed.
 
**Main > Settings > Web Client** tab

From this tab you can install and manage the VisualCron Web Client, either under **IIS** or **IIS Express**. For each option the tab shows whether IIS and the Web Client are installed, the *Port*, the *Link* to reach it, the *Source path* and whether *Windows authentication* is used, along with *Install*, *Uninstall* and *Enable* buttons.

See the [Web Client](../../using-visualcron/visualcron-web-client) section for more information.
