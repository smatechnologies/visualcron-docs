## Server

The **Server** tab is a part of the main menu. It is organized into the following groups:

* **Main settings** - [Server On/Off](server-on-off), [Settings](main-settings), [User permissions](main-user-permissions), Group permissions and [Client connections](main-client-connections).
* **Health** - opens the Server health window.
* **SLA** - [Uptime, Outages and Latency](sla-uptime-outages-latency).
* **Jobs** - Failed Jobs, Processes and [ROI](roi-return-on-investment).
* **Global objects** - the global objects such as Credentials, Connections, Notifications and Variables.
* **Information** - license and server information.
