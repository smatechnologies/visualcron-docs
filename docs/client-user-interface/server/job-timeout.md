## Job - Timeout

The timeout value can be set both at Job and Task level, for Task level see [Task - Timeout](../../client-user-interface/server/task-timeout) for more information.
 
**Add/Edit Job > Timeout** tab

![](../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Task%20Main%20Settings/TimeOut.png)

Check *Use timeout* and specify the time in hours, minutes and seconds. When the timeout is reached, all processes related to the current Job will be terminated.
