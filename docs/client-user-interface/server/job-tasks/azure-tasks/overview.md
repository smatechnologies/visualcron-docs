## Overview

Microsoft Azure (formerly Windows Azure) is a cloud computing service created by Microsoft for building, testing, deploying, and managing applications and services through a global network of Microsoft-managed data centers.

The Azure Tasks in VisualCron use a [Microsoft Azure Connection](../../connection-microsoft-azure) to manage virtual machines and web instances in the Azure cloud.

**Available Azure Tasks**

Virtual machines:

* [Create VM](create-vm)
* [Start VM](start-vm)
* [Stop VM](stop-vm)
* [Restart VM](restart-vm)
* [Update VM](update-vm)

Web sites:

* [Create Web Site](create-web-site)
* [Start Web Site](start-web-site)
* [Stop Web Site](stop-web-site)
* [Restart Web Site](restart-web-site)
* [Remove Web Site](remove-web-site)
