## Task Amazon EC2 - Terminate Instance

The Amazon EC2 - Terminate instance Task terminates an EC2 instance. Termination is permanent — the instance and its root EBS volume are deleted. To preserve the EBS volume for later reuse, use [Stop Instance](stop-instance) instead.

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/Amazon%20EC2%20Tasks/Terminate%20Instance.png)

**Connection**

To use Amazon EC2 Tasks you need to create an Amazon [Connection](../../connection-amazon) first. Click the *Settings* icon to open the *Manage Connections* dialog.

**Instance**

The instance ID to terminate. Click the *Refresh* icon to populate the drop-down instance ID list.
