## Overview

The Portable Document Format (PDF) is a file format used to present documents in a manner independent of application software, hardware, and operating systems.

The PDF Tasks in VisualCron let you read, modify, sign, encrypt and convert PDF files. Most PDF Tasks share a *File filter* sub tab to choose the source file(s) and a *Destination* sub tab to control where the result is written.
