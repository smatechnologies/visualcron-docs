## Command - Rename Folder

The Rename folder command renames a folder on the remote server.
 
**Rename folder** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/Net%20Tasks/Rename%20folder.png)

**Original name**

The path of the folder to rename on the remote server. Use the folder browse button to select a folder from the server.

**New name**

The new name for the folder.

**Credential**

Select a Credential if the folder resides on a network drive that requires authentication.
