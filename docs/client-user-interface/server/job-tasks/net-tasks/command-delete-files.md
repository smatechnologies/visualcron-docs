## Command - Delete Files

For delete files, VisualCron uses the [remote file filter](../../../server/job-tasks-remote-file-filter) to specify files that should be deleted.
 
**Delete file** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/Net%20Tasks/Delete%20File.png)

**Connection**

Select the connection to use for accessing the remote server. The Credential field is also shown and allows you to optionally specify a credential for authentication.

**Delete File**

The Delete file tab has no additional configuration fields. All files to be deleted are specified using the Remote file filter tab. The delete operation is performed on every file that matches the filter criteria.
