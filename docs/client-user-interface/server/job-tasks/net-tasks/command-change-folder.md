## Command - Change Folder

The Change folder command changes to another folder location on the remote server.
 
**Change folder** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/Net%20Tasks/FTP%20SFTP%20Command%20Types.png)

**Folder**

The path to change to on the remote server. Default is `/`. If the server supports it you can specify a relative path (e.g. `/folder/subfolder`); otherwise use multiple Change folder commands to navigate step by step. Use the folder browse button to select a folder from the server.
