## Command - List Files

For list files, VisualCron uses the [remote file filter](../../../server/job-tasks-remote-file-filter) to specify files that should be listed.

**List files** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/Net%20Tasks/List%20Files.png)

**Output**

**Full path**

When checked, the full path is included in the output for each listed file. When unchecked, only the file name is listed. Default is checked.
