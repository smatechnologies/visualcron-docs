## Overview

Microsoft Message Queuing (MSMQ) is a message queue implementation developed by Microsoft and shipped with Windows Server and Windows client operating systems.

The MSMQ Tasks in VisualCron let you work with queues on the local computer or on a remote computer. The connection settings (*Local MSMQ* / *Remote MSMQ*, *Credential*, *Host name* and *Queue*) are shared by all MSMQ Task types.

**Available MSMQ Tasks**

* [Send Message](send-message)
* [Delete Message](delete-message)
