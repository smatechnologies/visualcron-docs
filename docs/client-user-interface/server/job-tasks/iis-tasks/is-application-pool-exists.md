## Task IIS - Is Application Pool Exists

The IIS - Is Application Pool Exists Task checks whether a specific application pool exists in Internet Information Services (IIS). This Task returns a result indicating the presence of the application pool and can be used as a condition to control Job flow before performing further IIS operations.

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/IIS%20Tasks/Is%20Application%20pool%20exists.png)

**Connection**

To use IIS Tasks you need to create a [Connection](../../global-connections) first. Click the *Settings* icon to open the *Manage Connections* dialog.

**Application pool name**

The name of the application pool to check for existence.
