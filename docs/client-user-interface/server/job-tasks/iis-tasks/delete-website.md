## Task IIS - Delete Website

The IIS - Delete Website Task removes an existing website from Internet Information Services (IIS).

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/IIS%20Tasks/Delete%20an%20existing%20web%20site.png)

**Connection**

To use IIS Tasks you need to create a [Connection](../../global-connections) first. Click the *Settings* icon to open the *Manage Connections* dialog.

**Web site name**

The name of the website to delete.
