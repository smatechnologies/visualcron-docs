## Task IIS - Change Website State

The IIS - Change Website State Task starts, stops, or restarts an existing website in Internet Information Services (IIS).

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/IIS%20Tasks/Change%20state%20of%20the%20web%20site.png)

**Connection**

To use IIS Tasks you need to create a [Connection](../../global-connections) first. Click the *Settings* icon to open the *Manage Connections* dialog.

**Web site parameters**

**Web site name**

The name of the website to manage. Click the *Refresh* icon to populate the drop-down list with available websites.

**Operation**

The operation to perform on the website. Available options are:
- *Start* – Starts the website so it begins accepting requests
- *Stop* – Stops the website so it no longer accepts requests
- *Restart* – Stops and then starts the website
