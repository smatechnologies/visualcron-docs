## Task IIS - Delete Application Pool

The IIS - Delete Application Pool Task removes an existing application pool from Internet Information Services (IIS).

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/IIS%20Tasks/Delete%20an%20existing%20app%20pool.png)

**Connection**

To use IIS Tasks you need to create a [Connection](../../global-connections) first. Click the *Settings* icon to open the *Manage Connections* dialog.

**Application pool name**

The name of the application pool to delete.
