## Task System - Windows Update

The Windows update Task performs a Windows update in the background. You can select if you want to download updates only or install them as well.
 
![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/System%20Tasks/Windows%20Update.png)

 **Credentials**

To control a remote computer you may need to use a Credential. The Credential must match the user name and password of the user that you want to login for. Select a Credential in the combo box or click the *Settings* icon to open *Manage credentials* in order to add or edit Credentials.

**Shutdown settings**

**Local** / **Remote**

Select **Local** to run Windows Update on the local computer. Select **Remote** to run Windows Update on a remote computer. Defaults to **Local**.

**Hostname**

Enter the hostname of the remote computer, or select from the list of network computers. Only available when **Remote** is selected.

**Settings**

**Download updates**

When checked, downloads available Windows updates.

**Install updates**

When checked, installs downloaded Windows updates.

**Install hidden updates**

When checked, also installs updates that have been hidden. Only available when **Install updates** is checked.
