## Task System - Restart

The System restart Task can restart a local or remote computer.

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/System%20Tasks/Restart.png)

**Credentials**

To control a remote computer you may need to use a Credential. The Credential must match the user name and password of the user that you want to login for. Select a Credential in the combo box or click the *Settings* icon to open *Manage credentials* in order to add or edit Credentials.

**Shutdown settings**

**Local** / **Remote**

Select **Local** to restart the local computer. Select **Remote** to restart a remote computer.

**Hostname**

The hostname of the remote computer to restart. Select from the list of available network computers or type a hostname directly. Only available when **Remote** is selected.
