## Task VMWare - List VM Snapshots

The VMWare - List VM snapshots Task returns a list of existing snapshots of a virtual machine.

**List snapshots > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/List%20snapshots.png)

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.
