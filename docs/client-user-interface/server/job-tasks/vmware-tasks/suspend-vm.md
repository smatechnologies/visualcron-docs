## Task VMWare - Suspend VM

The VMWare - Suspend VM Task suspends a virtual machine.

**Suspend VM > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Suspend%20VM.png)

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.
