## Task VMWare - [Guest] Capture Screen

The VMWare - Capture screen Task takes a screenshot of the guest operating system and saves it to a file on the VisualCron server.

**[Guest] Capture screen > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Guest%20Section/Guest%20Capture%20Screen.png)

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.

**[Guest] Capture screen > VM Task settings** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Guest%20Section/Guest%20Capture%20Screen%20Settings.png)

**Image path**

Full path to the PNG file where the captured screen should be saved. Click the *File* icon to choose the destination file.
