## Task VMWare - Install Tools

The VMWare - Install tools Task installs the VMWare Tools package in the guest operating system.

**Install tools > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Install%20Tools.png)

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.
