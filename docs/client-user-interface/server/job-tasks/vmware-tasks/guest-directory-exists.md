## Task VMWare - [Guest] Directory Exists

The VMWare - Directory exists Task returns True or False depending on whether a directory exists in the guest operating system.

**[Guest] Directory exists > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Guest%20Section/Guest%20Directory%20Exists.png)

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.

**[Guest] Directory exists > VM Task settings** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Guest%20Section/Guest%20Directory%20Exists%20Settings.png)

**Source folder**

The path of the folder to check for in the guest operating system.
