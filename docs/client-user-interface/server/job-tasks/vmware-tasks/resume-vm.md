## Task VMWare - Resume VM

The VMWare - Resume VM Task resumes a paused virtual machine. This Task uses the *Local VMWare workstation* connection type.

**Resume VM > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Resume%20VM.png)

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.
