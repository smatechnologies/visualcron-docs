## Task VMWare - Delete VM

The VMWare - Delete VM Task lets you delete a virtual machine. This Task uses the *Remote VMWare server* connection type.

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Delete%20VM.png)

**Delete VM > Connection** sub tab

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.
