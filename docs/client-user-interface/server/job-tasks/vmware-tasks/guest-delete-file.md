## Task VMWare - [Guest] Delete File

The VMWare - Delete file(s) Task deletes one or more files in the guest operating system.

**[Guest] Delete file(s) > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Guest%20Section/Guest%20Delete%20Files.png)

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.

**[Guest] Delete file(s) > VM Task settings** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Guest%20Section/Guest%20Delete%20Files%20Settings.png)

**Source folder**

The folder to delete files from.

**Source file (mask)**

The file name or wildcard mask matching the files to delete.
