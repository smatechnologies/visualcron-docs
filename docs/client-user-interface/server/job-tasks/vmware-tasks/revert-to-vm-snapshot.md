## Task VMWare - Revert to VM Snapshot

The VMWare - Revert to VM snapshot Task restores the virtual machine to the state of a specific snapshot.

**Revert to snapshot > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Revert%20to%20Snapshot.png)

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.

**Revert to snapshot > VM Task settings** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Revert%20to%20Snapshot%20settings.png)

**Snapshot name**

The name of the snapshot to revert to. Click the *Refresh* icon to populate the drop-down snapshot list.
