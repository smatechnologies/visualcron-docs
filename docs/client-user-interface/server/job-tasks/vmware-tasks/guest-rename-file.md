## Task VMWare - [Guest] Rename File

The VMWare - Rename file Task renames a file in the guest operating system.

**[Guest] Rename file > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Guest%20Section/Guest%20Rename%20File.png)

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.

**[Guest] Rename file > VM Task settings** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Guest%20Section/Guest%20Rename%20File%20Settings.png)

**Old file path**

The full path of the file to rename, as it exists today.

**New file path**

The new full path for the file.
