## Task VMWare - [Guest] Open URL

The VMWare - Open URL Task opens an URL on the guest operating system.
 
**[Guest] Open URL > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Guest%20Section/Guest%20Open%20URL.png)

The Connection settings window is the same for all VMWare Task types. See [Start VM](start-vm) for details.
 
**[Guest] Open URL > VM Task settings** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Guest%20Section/Guest%20Open%20URL%20Settings.png)

**URL**

The URL you want to open.
