## Task VMWare - Pause VM

The VMWare - Pause VM Task puts a virtual machine in pause mode. This Task uses the *Local VMWare workstation* connection type.

**Pause VM > Connection** sub tab

![](../../../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Jobs/Job%20Tasks/Tasks/VMWare%20Tasks/Pause%20VM.png)

The Connection sub tab is the same for all VMWare Task types. See [Start VM](start-vm) for details.
