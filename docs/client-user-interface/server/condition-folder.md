## Condition - Folder

The *Folder* condition lets you check whether a folder exists before running a Task or Job.

**Global > Conditions > Condition set Add > Conditions Add > Folder select > Folder** tab

![](../../../static/img/Client%20User%20Interface/Main%20Menu/Server/Global%20Objects/Global%20-%20Conditions/Condition%20-%20Folder.png)

**Folder path**

The full UNC path to the folder to check. Click the *Folder* browser icon to the right to browse for the folder.

**Credentials**

If the folder is on a network share or any other protected location, select a [Credential](global-credentials) for accessing it. Click *Manage credentials* to add a new Credential.

**Folder exists**

Checks whether the folder exists. Select *True* or *False* to match.
